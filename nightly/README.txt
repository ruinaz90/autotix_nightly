# AutoTix Nightly
An application to automate nightly for the fulfillment team.

## Installation
Use the package manager to install AutoTix Nightly.

## Instructions
1. Make sure you have pulled a report on CMS and saved the file as a CSV to your Downloads folder.

2. Open the app and click Run. A pop-up should appear when the process is complete.

3. Close the application.

4. Open the Nightly Check-in doc on Google Sheets.

5. Paste the information in cell A1.
